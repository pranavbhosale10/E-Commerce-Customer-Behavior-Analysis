# 🛒 E-Commerce Customer Behavior Analysis

## 📌 Project Overview

This project presents an end-to-end Power BI dashboard built using the Brazilian E-Commerce (Olist) dataset. It analyzes customer purchasing behavior, revenue trends, product performance, and business KPIs to generate actionable business insights.

---

## 🎯 Objectives

- Analyze customer purchase behavior
- Identify top-performing products and categories
- Monitor revenue and order trends
- Track customer retention and repeat purchases
- Build interactive dashboards for business decision-making
